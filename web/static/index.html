📖 N / R / D / B 如何组合成最终 prompt（点开看清单）
四个变量
  • N = 自然语言框（-t）。是否调 LLM 完全看 N 有没有填。
  • R = 改写模式（-r）。仅在 N 非空时生效。
  • D = 直接 Tag 框（-d），手写英文 tag。
  • B = 工作流内置 prompt，上面那个只读框。
统一记号：底本 = B , D（按顺序拼接，空的跳过）
勾选「丢弃内置 B」后：底本 = D，B 不参与任何分支。
三种情况
条件 LLM 做什么 最终 prompt
N 空 不调用 底本
N 有 + R 关 把 N 翻成英文 tag 底本 , 翻译(N)
N 有 + R 开 把底本当原始 prompt，按 N 整体重写
底本为空时降级为翻译		 改写输出
只有未勾选「丢弃内置 B」时，B 才会参与。改写模式下 LLM 会智能修改/删除/补充底本，最终只输出重写后的内容。
